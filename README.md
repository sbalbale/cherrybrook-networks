## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/sbalbale/cherrybrook-networks

# Access
$ cd cherrybrook-networks

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
